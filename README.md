# LogSim Credibility Audit

Research workspace for auditing the credibility of log-driven counterfactual closed-loop simulation for autonomous driving.

## Current focus

The current first audit target is **OmniDreams**.

This repository should not become a broad simulator survey at the beginning. The first goal is to understand how OmniDreams proves, or fails to prove, the credibility of its own simulation results.
